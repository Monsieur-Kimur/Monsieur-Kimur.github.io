Проекты

Создание универсальных методических пособий для обучения программированию в визуально-блочных системах

Я разработал универсальные методические пособия для освоения техник программирования в визуально-блочных системах и успешно защитил проект основанный на этом. Основной его идей было то, что человек в любом возрасте способен реализовывать свои идеи через програмный код, а наилучшим стартом для этого являются визуально-блочные системы, по типу Scratch

Платформа для онлайн-соревнований по программированию

Создал веб-платформу, которая позволяет организаторам проводить соревнования по программированию в реальном времени. Решение успешно использовалось в нескольких школах и привлекло не менее сотни участников.

Мобильное приложение для изучения иностранных языков

Участвовал в разработке приложения, которое использует геймификацию для повышения эффективности изучения новых языков. Приложение было загружено более 50,000 раз и получило положительные отзывы пользователей.
